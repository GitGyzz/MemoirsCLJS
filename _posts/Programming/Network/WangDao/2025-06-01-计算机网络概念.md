# 计算机网络概念

计算机网络:由若干节点node 和 链路 link
组成(由交换机，集线器或路由器组成的网络)

> 节点
>
> > 计算机
> >
> > 集线器
> >
> > > 会数据冲突,不太好用,大约被淘汰
> >
> > 交换机
> >
> > > 不会发生数据冲突,组成计算机网络,集线器的升级版
> >
> > 路由器
> >
> > > 可以把两个或多个计算机网络互相连接起来.形成互连网.
> >
> > 家用路由器
> >
> > > 路由器+交换机+其他功能
>
> 链路 有线和无线

互连网:没连上因特网 可以使用任意协议

互联网:Internet,全球范围 必须使用TCP/IP协议

ISP:Internet Service Provider 互联网服务提供商
